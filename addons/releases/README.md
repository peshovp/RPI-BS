## Viewing Releases

### GitHub Releases Page
https://github.com/peshovp/GeoMaxima-BS/releases

### Using GitHub CLI
```bash
# List all releases
gh release list

# View specific release
gh release view v1.3.1

# Open in browser
gh release view v1.3.1 --web
```

### Using Git Tags
```bash
# List all tags
git tag -l

# Show tag message
git show v1.3.1

# Checkout specific version
git checkout v1.3.1
```
