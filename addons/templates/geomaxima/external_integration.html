About
What does this do?

GNSS.NET Caster Integration allows your base station to:

  • Register with central caster
  • Report health and status
  • Update coordinates automatically after Auto Survey
  • Receive remote configuration
  • Enable centralized monitoring
API Endpoints Used:
  • POST /api/stations/{id}/status - Send status updates
  • GET /api/stations/{id}/config - Receive configuration (future)
Auto Survey

When enabled, coordinates will be automatically sent to GNSS.NET caster after Auto Survey completion.

Ensure caster integration is enabled before running Auto Survey
