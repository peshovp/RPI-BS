RTCM Message Presets

Reference guide for RTCM 3.x message types and configurations

Quick Guide: Use the dropdown menu in Settings → Ntrip A/B service to quickly select optimized RTCM message combinations.
MSM4 - Compact

Bandwidth: 300-500 bytes/s

Accuracy: 2-3 cm RTK Fix

Best for:

  • 4G/LTE connections
  • RTK2GO network
  • Fast update rates

Messages:

1074 (GPS), 1084 (GLONASS), 1094 (Galileo), 1124 (BeiDou)
MSM5 - Full Precision ⭐

Bandwidth: 500-800 bytes/s

Accuracy: 1-2 cm RTK Fix

Best for:

  • Centipede network
  • General purpose base
  • Balanced performance

Messages:

1075 (GPS), 1085 (GLONASS), 1095 (Galileo), 1125 (BeiDou)
MSM7 - High Resolution

Bandwidth: 800-1200 bytes/s

Accuracy: 0.5-1 cm RTK Fix

Best for:

  • Professional surveying
  • Local network (LAN)
  • Maximum accuracy

Messages:

1077 (GPS), 1087 (GLONASS), 1097 (Galileo), 1127 (BeiDou)
CMR Enhanced Variants

CMR (Compact Measurement Record) variants add ephemeris data for compatibility with legacy GNSS receivers.

Preset Additional Messages Purpose
MSM4 + CMR 1019, 1020, 1042, 1045, 1046 Compact + Legacy support
MSM5 + CMR 1019, 1020, 1042, 1045, 1046 Full precision + Legacy
MSM7 + CMR 1019, 1020, 1042, 1045, 1046 Maximum compatibility
CMR Messages:
  • 1019: GPS Ephemeris
  • 1020: GLONASS Ephemeris
  • 1042: BeiDou Ephemeris (D1 NAV)
  • 1045: Galileo F/NAV Ephemeris
  • 1046: Galileo I/NAV Ephemeris
Base Station Messages (Common to All)
Message Description Interval
1005 Station coordinates (ECEF) 10 seconds
1006 Station coordinates + antenna height On demand
1033 Receiver and antenna descriptors 10 seconds
1230 GLONASS L1 and L2 code-phase biases 1 second
Which Preset Should I Use?
Use Case Recommended Preset Reason
Centipede Network MSM5 or MSM5 + CMR Balanced accuracy and bandwidth
RTK2GO MSM4 Lower bandwidth for public casters
Local Network (LAN) MSM7 Maximum accuracy, bandwidth not limited
4G/LTE Connection MSM4 Minimize mobile data usage
Legacy Receivers MSM5 + CMR Ephemeris for older GNSS receivers
Professional Surveying MSM7 + CMR Maximum accuracy and compatibility
Custom Configuration

If you need a custom RTCM message combination:

  1. Go to Settings → Ntrip A/B service
  2. Select "Custom (Manual Entry)" from the dropdown
  3. Enter your custom message list in the format: msg1,msg2(interval),msg3,...

Example:

1005(30),1077,1087,1097,1127

This sends station coordinates every 30 seconds and MSM7 messages for all constellations every second.
